Responsive Web Design Principles

- Media queries allow website content to be viewed differnetly on different devices by changing viewport sizes

@media(min-width: 1000px) { p{font-size: 20px} }

- img styling will make img responsive

- 4 units for viewports

  1. vw: 10vw - means that content is 10% of viewport width
  2. vh: 20vh - means that content is 10% of viewport height
  3. vmin: 50vmin - means that content is 10% of viewport's smaller dimention (w h)
  4. vmax: 100vmax - means that content is 10% of viewport's larger dimention (w h)

    Data Structures

    Arrays

    - bad for inserting new data

    - bad for deleting data

    - data lookup is good, random access and constant time

    - easy to sort

    - reletively small data type

    - not flexible, once you initialize, size can't be changed

    Linked Lists

    - easy to insert new data

    - easy to delete data

    - data lookup is bad, linear lookup

    - hard to sort

    - great with small size

    Hash Table

    - insertion is a 2-step process, hash + add

    - easy to delete data

    - lookup is better than linked lists

    - not good for sorting, array is better for this

    - size depends

    Tries

    - complex to insert data

    - easy to delete

    - fast lookup

    - already sorted

    - very large size
