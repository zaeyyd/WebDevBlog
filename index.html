Forms

(text forms)

- get input with "input" ELEMENT

- types of input

- "placeholder" in input

- "action" in form

- "required" in input

(form action="url to send data")
(input type="text" placeholder="text")
(button type="submit")this button submits the form(/button)
(/form)

(radio forms)

- "radio" is a type of input

- "label" ELEMENT for text around button

- "name" ATTRIBUTE in form makes it so that radio is automatically deselected when another radio of the same name is selected

- "id" ATTRIBUTE in input for unique radios

- "for" ATTRIBUTE in label is to accociate label to input id

- "checked" ATTRIBUTE for radio

(label for="BOB") First Radio
(input id="First" type="radio" name="BOB")
(/label)

(label for="BOB") Second Radio
(input id="Second" type="radio" name="BOB")
(/label)
